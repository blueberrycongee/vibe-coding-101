# Vibe Coding 101

[English](README.md) | 简体中文 | [日本語](README.ja.md) | [한국어](README.ko.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Français](README.fr.md) | [Deutsch](README.de.md)

一个系统化的 vibecoding 教程仓库：收集名人技巧，提炼为可复用的方法，并融入实际工作流。

## 目标
- 帮助新人理解 vibecoding 的核心心智模型
- 把分散的技巧沉淀为流程与模板
- 形成可持续迭代的教程与案例库

## 适合谁
- 想系统化学习 vibecoding 的工程师或设计师
- 想把“灵感型写法”落地为稳定流程的团队

## 学习路径（建议顺序）
1. `docs/00-overview.md`：概念、范围、边界
2. `docs/01-foundations.md`：基础原则与常见误区
3. `docs/02-workflow.md`：端到端流程框架
4. `docs/03-prompts.md`：提示词与任务拆解
5. `docs/04-critique.md`：复盘、审查与迭代
6. `docs/05-collaboration.md`：多人协作与规范
7. `docs/06-case-studies.md`：案例拆解
8. `docs/07-tools.md`：工具栈与实践
9. `docs/08-ethics.md`：引用、版权与责任
10. `docs/09-roadmap.md`：后续路线图

## 仓库结构
- `docs/`：线性教程主线
- `notes/`：原子技巧条目（标签 + 双链）
- `indexes/`：索引页（按人、语言、主题聚合）
- `examples/`：短例子与可运行小样
- `tools/`：好用的 skills 与 MCP 归档
