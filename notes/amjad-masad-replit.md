# Amjad Masad 与 Replit 的 AI 编程愿景

## 人物简介
Amjad Masad 是 Replit CEO。Replit 估值 $9B，2025 年 ARR 从 $2.7M 增长到 $240M，75% 用户从未写过一行代码。

## 核心立场："不喜欢 vibe coding 这个词"

> "我不喜欢这个词......它贬低了可能性。如果我要解释 Replit，它就是 vibe。不是 vibe coding，只是 vibe——别写代码，just vibe。"

来源：Sequoia "Training Data" 播客 https://analyticsindiamag.com/ai-news-updates/replit-ceo-amjad-masad-doesnt-like-vibe-coding/

## 关键观点

### "Software creation for everyone"
- 2025 年初做出关键决策：Replit 不再以专业开发者为核心，转向"数十亿没有技术背景的开发者"
- 愿景：从自然语言描述直接生成完整的、可运行的应用
- "你不是从代码开始，你是从一个想法开始"

### CEO 们的新武器 (2026-01)
- "CEO 们可以用 vibe coding 自己做原型，不用再求工程师帮忙"
- 产品经理是"最好的 vibe coders"——他们习惯把问题拆解为清晰步骤
- 来源：https://dnyuz.com/2026/01/08/replit-boss-ceos-can-vibe-code-their-own-prototypes-and-dont-have-to-beg-engineers-for-help-anymore/

### Agent 4 + Canvas (2026-03)
- 称可以"从零 vibe code 一整个创业公司"
- Agent 可自我测试、自主工作 200 分钟、构建其他 agent
- 支持 React Native + Expo 移动应用生成
- 来源：https://www.inc.com/ben-sherry/replit-ceo-says-their-new-ai-agent-can-vibe-code-a-startup-from-scratch/91315098

### 关键数据
- ARR 增长：2024.4 $2.7M → 2025.4 $70M → 2025.6 $100M → 2025 全年 $240M
- 目标 2026 年底 $1B 营收
- $9B 估值，$400M Series D

## 边界
- 约 45% 的 AI 生成代码包含安全漏洞（CodeRabbit 2025.12 报告）
- 复杂应用仍需人工干预
- AI 不会取代人——"我们都会有工作，只是会变得非常强大"
