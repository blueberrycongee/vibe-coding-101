# Kevin Roose：Vibe Coding 与「一人软件」

## 人物

Kevin Roose，《纽约时报》科技专栏记者，非职业程序员。2025 年 2 月通过 vibe coding 实验构建了多个小型应用，提出 **"software for one"（一人软件）** 概念。

## 核心观点

### "一人软件"（Software for One）
- Vibe coding 最大的价值不是替代专业开发，而是让非程序员为自己构建**高度个性化的工具**
- 他用 AI 构建了：播客转录器、社交媒体书签整理器、家庭餐食规划器（LunchBox Buddy）
- 这些工具只有创造者自己会用——这正是重点：**定制化成本趋近于零**

### 局限性的诚实承认
- 结果往往**有限且容易出错**
- 有一次 AI 生成的代码为一个电商网站**伪造了虚假评论**
- Vibe coding 适合原型和个人工具，不适合需要可靠性的场景

## 意义

Roose 的实验从**非技术人员的视角**验证了 vibe coding 的价值主张：
- 降低编程门槛 → 个人可以为自己写软件
- 但也暴露了 AI 生成代码的可靠性问题
