## 工作流技巧

| 技巧 | 说明 |
|------|------|
| 只用语音输入 | 用 Whispr Flow 语音转文字，配合 Cursor Composer |
| 从模板开始 | 使用预构建的 GitHub 模板（带 Firebase 数据库和认证） |
| 用 Perplexity 学 API | 将 Perplexity 作为研究工具理解和实现 API |
| 管理上下文长度 | 保持 Composer 线程小，谨慎管理上下文 |
| 先画再做 | 用 Whimsical 可视化想法，用 Claude 生成 Mermaid 图 |
| 在 Cursor 中添加文档 | 让 AI 可以引用文档实现功能 |
| 部署 | Vercel + GitHub + NameCheap 域名 |

## 愿景
- 把 App 开发变得像 **Canva 做设计**一样简单
- VibeCode App 已有 200 名 Beta 用户创建了 3000+ 应用

## 边界
- 目标受众是非程序员，不适合有经验的开发者做生产开发
- 语音优先在嘈杂环境或需要精确技术术语时效果差
- 无代码审查能力意味着安全和质量风险
