# Vibe Coding 101

[English](README.md) | [简体中文](README.zh-CN.md) | 日本語 | [한국어](README.ko.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Français](README.fr.md) | [Deutsch](README.de.md)

vibecoding を体系的に学ぶためのリポジトリです。著名人の手法を収集し、再利用可能な方法に整理して、実践的なワークフローに統合します。

## 目的
- vibecoding の中核となる思考モデルを理解する
- 断片的なコツをプロセスやテンプレートに落とし込む
- 教材と事例を継続的に更新できる形で蓄積する

## 対象
- vibecoding を体系的に学びたいエンジニア/デザイナー
- ひらめき頼みの作業を安定したフローにしたいチーム

## 学習順序（推奨）
1. `docs/00-overview.md` — 概念、範囲、境界
2. `docs/01-foundations.md` — 基本原則と落とし穴
3. `docs/02-workflow.md` — エンドツーエンドの流れ
4. `docs/03-prompts.md` — プロンプトとタスク分解
5. `docs/04-critique.md` — レビューと改善
6. `docs/05-collaboration.md` — 協業と規範
7. `docs/06-case-studies.md` — 事例
8. `docs/07-tools.md` — ツールと実践
9. `docs/08-ethics.md` — 引用・著作権・責任
10. `docs/09-roadmap.md` — ロードマップ

## リポジトリ構成
- `docs/` — チュートリアル本編
- `notes/` — 原子化テクニック（タグ + バックリンク）
- `indexes/` — 索引（人物・言語・トピック）
- `examples/` — 再現可能な小さな例
- `tools/` — skills と MCP の整理
