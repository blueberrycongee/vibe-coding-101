# Vibe Coding 101

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | 한국어 | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Français](README.fr.md) | [Deutsch](README.de.md)

vibecoding을 체계적으로 배우기 위한 저장소입니다. 유명 인물의 기법을 수집해 재사용 가능한 방법으로 정리하고, 실전 워크플로에 통합합니다.

## 목표
- vibecoding의 핵심 사고 모델 이해
- 흩어진 팁을 프로세스와 템플릿으로 정리
- 튜토리얼과 사례를 지속적으로 축적

## 대상
- vibecoding을 체계적으로 배우고 싶은 엔지니어/디자이너
- 영감 중심 작업을 안정적 흐름으로 바꾸고 싶은 팀

## 학습 순서(권장)
1. `docs/00-overview.md` — 개념, 범위, 경계
2. `docs/01-foundations.md` — 기본 원칙과 흔한 실수
3. `docs/02-workflow.md` — 엔드 투 엔드 워크플로
4. `docs/03-prompts.md` — 프롬프트와 작업 분해
5. `docs/04-critique.md` — 리뷰, 비평, 개선
6. `docs/05-collaboration.md` — 협업과 규범
7. `docs/06-case-studies.md` — 사례
8. `docs/07-tools.md` — 도구와 실천
9. `docs/08-ethics.md` — 인용, 저작권, 책임
10. `docs/09-roadmap.md` — 로드맵

## 저장소 구조
- `docs/` — 선형 튜토리얼 본문
- `notes/` — 원자적 기법(태그 + 백링크)
- `indexes/` — 인덱스(인물, 언어, 주제)
- `examples/` — 재현 가능한 작은 예시
- `tools/` — skills 및 MCP 정리
