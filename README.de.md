# Vibe Coding 101

[English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Français](README.fr.md) | Deutsch

Ein strukturiertes Repository zum Lernen von vibecoding: Wir sammeln Methoden bekannter Personen, destillieren sie zu wiederverwendbaren Verfahren und integrieren sie in einen praktischen Workflow.

## Ziele
- Einsteiger*innen helfen, die Kernmodelle des vibecoding zu verstehen
- Verstreute Tipps in Prozesse und Vorlagen überführen
- Eine lebende Bibliothek aus Tutorials und Fallstudien aufbauen

## Für wen
- Entwickler*innen oder Designer*innen, die vibecoding systematisch lernen möchten
- Teams, die Inspiration in stabile, wiederholbare Abläufe überführen wollen

## Repository-Struktur
- `docs/` — Linearer Tutorial-Pfad
- `notes/` — Atomare Techniken (Tags + Backlinks)
- `indexes/` — Indizes (Personen, Sprachen, Themen)
- `examples/` — Kleine reproduzierbare Beispiele
- `tools/` — skills und MCP-Referenzen
